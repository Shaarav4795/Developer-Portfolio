# Shaarav Arya Portfolio

Personal portfolio built with React, TypeScript, GSAP, and Three.js.

## Development

- `npm install`
- `npm run dev`
- `npm run build`

## Blog Authoring

- Create posts in `src/content/posts/`
- Use `src/content/posts/_template.md` as the starter
- Full guide: `src/content/posts/README.md`
